# CryptoSafe-Manager
– Кроссплатформенный менеджер паролей с графическим интерфейсом, зашифрованной локальной базой данных и защищенной работой с буфером обмена.



<font size="4">*Видение проекта*</font>

На первом этапе (Спринт 1) создается безопасный, расширяемый фундамент с:
Зашифрованной базой данных (заглушка шифрования через AES XOR)
Модульной архитектурой (разделение Core/GUI/Database)
Базовой оболочкой графического интерфейса (PyQt6, main_window и виджеты-заглушки)

Дальнейшие функции будут внедряться в следующих спринтах.



<font size="4">*|   Спринт   |   Цель   |*</font>

| Спринт 1 | Создание базового фундамента: структура, криптозаглушки, база данных, GUI-заглушка |

| Спринт 2 | Реализация настоящего KeyManager и системы ключей |

| Спринт 3 | Реальное AES-GCM шифрование и замена заглушек |

| Спринт 4 | Буфер обмена с таймером и очистка, улучшенные события |

| Спринт 5 | Журнал аудита и просмотр логов |

| Спринт 6 | Импорт/экспорт записей, интеграция тегов |

| Спринт 7 | Авто-блокировка и таймер неактивности |

| Спринт 8 | Резервное копирование, финальная сборка и Docker |
