# Установка и запуск

## Требования

- Python 3.12+
- pip
- virtualenv / venv
- Git

## 1. Клонирование репозитория

```bash
git clone https://github.com/your-username/cryptosafe-manager.git

cd cryptosafe-manager
```

## 2. Создание виртуального окружения

```bash
python -m venv venv
```

### Активация окружения

#### Linux / macOS

```bash
source venv/bin/activate
```

#### Windows

```powershell
venv\Scripts\activate
```

## 3. Обновление pip

```bash
python -m pip install --upgrade pip
```

## 4. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 5. Запуск приложения

```bash
python src/app.py
```

## 6. Запуск тестов

```bash
pytest tests/
```

## Пояснение архитектуры

Приложение построено по многоуровневой архитектуре,
где каждый слой отвечает за собственную область логики.

- **GUI Layer** — пользовательский интерфейс приложения.
- **Application Layer** — обработка событий, управление состоянием и взаимодействие между компонентами.
- **Core Services** — основная бизнес-логика password manager.
- **Security Layer** — шифрование, управление ключами и защита данных.
- **Database Layer** — хранение данных и работа с SQLite.

При работе приложения пользователь взаимодействует с GUI,
после чего запросы передаются через Application Layer
во внутренние сервисы и Security Layer.

Перед сохранением данные шифруются,
после чего сохраняются в локальную базу данных.
